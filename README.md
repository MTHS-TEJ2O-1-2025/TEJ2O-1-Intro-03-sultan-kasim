## To lint your python code

Run the following command in the terminal, where your *.py file is located:

```console
black --check --diff ./*.py
```
## To Build *.hex for Micro:bit using MicroPython

Run the following command in the terminal, where your *.py file is located:

``` bash
py2hex ./main.py
```
